# Create a Folder

* Bruno's folders
  * helps you
    * organize your API requests
      * recommendation: == your API's organization

## Creating a New Folder

1. | Bruno,
   * Collection > `···` > `New Folder`

    ![create folder](/public/screenshots/get-started/bruno-basics/create-folder/1-create-new-folder.webp)

     * Folder Name: Choose one

       ![new folder dialog](/public/screenshots/get-started/bruno-basics/create-folder/2-create-sample-folder.webp)

       * | Bruno v1.40.0+,
         * you can use special characters (`/`, `[`, `]`, `*`, etc)
           * if you use unsupported characters -> Bruno AUTOMATICALLY handles filesystem compatibility -- by replacing by -- `-`

## How to View or Customize Filesystem Names

* TODO:
1. Select **Show Filesystem Name.**

![show filesystem name](/public/screenshots/get-started/bruno-basics/create-folder/3-show-file-system-name.webp)

2. You can now see how your folder name is stored on the filesystem.

![filesystem name view](/public/screenshots/get-started/bruno-basics/create-folder/4-file-system-name.webp)

3. (Optional) Click the edit icon to customize the filesystem name while keeping your display name intact.

This feature allows you to:

- Create more descriptive folder organization.
- Use special characters freely.
- Maintain filesystem compatibility automatically.
- Keep your collection structure clean and organized.

### Example

You could name a folder "API [v2.0] / Authentication" and Bruno will automatically handle the filesystem compatibility while preserving your desired display name.
